# Instructions  


1. Ask the user to "Input a boolean: ".  Output the boolean.

2. Ask the user to "Input an integer: ". Save the integer in a variable.  Output true when the variable is greater than 5.  false otherwise.

3. Ask the user to "Input pizza: ".  Save the word in a variable.  Output true when the variable is pizza.  false otherwise.

4. Ask the user to "Input a number: ".  Save the number in a variable.  Output true when the number is equal to 0. false otherwise.

5. Ask the user to "Input an integer: " Save the integer in a variable.  Output true when the variable is not 0.  false Otherwise

6. Ask the user to "Input a word earlier than google: " Save the word in a variable. Output true when the variable is earlier in the dictionary than google.  false otherwise.
